# Roll a Ball 🎮

The Roll-a-Ball project is the classic starter tutorial for Unity, designed to teach the fundamentals of the editor, physics, and basic C# scripting.
The game involves a player-controlled ball that rolls around a play area to collect objects while staying within the boundaries.

## The Project 📁
The project includes more features than the classic Roll-a-Ball, such as:
- An enemy character (Wich follows the player, similar to Pac-Man gameplay).
- A Scoreboard.
- A win/loss system.
